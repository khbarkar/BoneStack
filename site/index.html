BoneStack

Container diagnostics, forensics, and optimization in a terminal UI. BoneStack can inspect images and containers, hunt for suspicious artifacts, diff filesystem changes, build a timeline from Docker events, export reports, and optionally use YARA during threat hunting.

GitHub Repo Quick Start Detailed Docs

Install

curl -fsSL https://raw.githubusercontent.com/khbarkar/BoneStack/main/install.sh | bash

After installation you can update BoneStack directly with:

bonestack update

Core Workflows

Container Forensics

Use f from container details to open filesystem, logs, processes, environment, resources, threat hunt, diff, and timeline views.

Training Examples

Build the safe demo containers from the repository root:

docker build -t bonestack-training:cron-shadow training/containers/cron-shadow
docker build -t bonestack-training:ssh-drop training/containers/ssh-drop-demo
docker build -t bonestack-training:bloaty-node training/containers/bloaty-node-demo

Run them:

docker run -d --name cron-shadow-demo bonestack-training:cron-shadow
docker run -d --name ssh-drop-demo bonestack-training:ssh-drop
docker run -d --name bloaty-node-demo bonestack-training:bloaty-node

Then open BoneStack, go to View Containers, select one of the demo containers, and press f for the forensics menu.

Cleanup:

docker rm -f cron-shadow-demo ssh-drop-demo bloaty-node-demo

What BoneStack Covers

  • image inspection and layer analysis
  • optimization reports in JSON, CSV, and HTML
  • Dockerfile and policy scaffolding
  • container threat hunt, diff, and timeline workflows
  • combined container forensics report export
